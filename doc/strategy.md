# 设计方案描述


## 技术架构
* 上层决策  决策支持系统
    * 判断何时何种情况做什么操作
    * 控制开启关闭某些中层行为
* 中层行为  
    * 多个行为模块完全解耦，互不关联
    * 遍历所有可能性，依据模块功能打分
    * 打分结果统一由决策模块处理
* 底层支持 黑板共享数据
